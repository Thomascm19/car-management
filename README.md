# car-management

# Proyecto final


car-management está construido usando las siguientes tecnologías 

- Angular
- .Net Core
- PostgreSQL
- Auth0 (Loggin)

## Funcionalidades

- Permite almacenar el perfil del usuario en la base de datos una vez este realice el login
- Permite actualizar la información del perfil
- Realizar la simulación y una vez esta es finalizada se guarda la información en la base de datos
- Permite visualizar la foto de perfil del usuario
- Como usuario admin, permite visualizar la lista de usuarios con toda su información.
